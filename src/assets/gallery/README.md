# Galeri Resimleri

Bu klasöre atölye galeri resimlerini ekleyin.

## Gerekli Dosyalar (500x357px veya benzer oran)

1. `turbo-galeri-1.webp` (veya .jpg)
2. `turbo-galeri-2.webp`

4. `turbo-galeri-4.webp`

## Boyut
- **Genişlik**: 500px (veya daha büyük)
- **Yükseklik**: 357px (veya benzer oran)
- **Format**: WebP (önerilen), JPG veya PNG
- **Aspect Ratio**: Yaklaşık 4:3 (1.4:1)

## Notlar
- Tüm resimler benzer boyut ve oranda olmalı
- Yüksek kaliteli, net görseller kullanın
- Web için optimize edilmiş dosyalar tercih edilir
- Mobilde 2, masaüstünde 4 sütun olarak gösterilecek
